Для интеграции с другими приложениями (такими как Tasker) предусмотрена обработка событий (Intents).
Для того, чтобы из таскера вызвать СитиГид с установленным маршрутом надо сделать задачу:
Разное - Send Intent
В поле действие надо указать: ru.shutoff.cgstarter.STARTER
В поле Extra надо указать: ROUTE:<номер машрута>
Номер маршрута - это номер предустановленной кнопки.
Если задать отрицательное значение, то СитиГид будет запущен без маршрута.
Значение 0 или без значения, указывает на то что надо запустить СитиГид с существующим маршрутом.
Также можно задать финиш в виде <широта>|<долгота>
